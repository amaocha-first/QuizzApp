# QuizzApp

![demo](https://github.com/amaocha-first/QuizzApp/blob/media/QuizzlerAppDemo.gif)

Udemy講座[iOS 11 & Swift 4 - The Complete iOS App Development Bootcamp](https://www.udemy.com/ios11-app-development-bootcamp/?start=0)で、制作したクイズアプリです。  
講座内ではクイズ内容が英語のため、日本語のクイズに変えました。



## このアプリで学習した内容
MVCの基礎を把握するために、QuestionクラスをModelとして作り、Controllerへ渡すなどしました。
